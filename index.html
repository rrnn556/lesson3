Работа с Git (команды)

  • Git --version - Команда показывает наличие программы на компьютере и её версию. Производится один раз.
  • git config --global user.name "Имя" - Имя пользователя в Git (для коммитов и работы с другими ветками) Производится один раз.
  • git config --global user.email "email пользователя" - e-mail пользователя в Git (для коммитов и работы с другими ветками) Производится один раз.
  • git init - Команда создаёт новый репозиторий. В проекте делается один раз
  • git remote add origin https://............ - Команда для связи Git c GitHub
  • git push origin ….. Команда показывает в какой ветке идет работа
  • git branch “test” – Команда для создания второй ветки прим. Test.
  • git branch – Команда покажет все ветки.
  • git branch -d ……. – Команда удаления ветки.
  • git marge ……… - Команда объединит две ветки.
  • git log – Команда просмотров коммитов.
  • git help - Здесь можно посмотреть все команды по работе с Git
